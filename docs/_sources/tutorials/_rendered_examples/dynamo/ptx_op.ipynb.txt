        "\n\n# Pre-compiled PTX kernels via ``torch_tensorrt.kernels.ptx_op``\n\n``cuda_kernel_op`` compiles your CUDA C++ source internally with NVRTC. When\nyou already have PTX in hand \u2014 emitted by Triton's JIT, cached from a prior\nNVRTC run on another machine, or hand-written \u2014 use ``ptx_op`` instead.
