C++ API
=======

C++ API reference for Torch-TensorRT, generated from source headers via Doxygen.

.. note::

   The C++ API is in maintenance mode. For new projects use the Python API via
   ``torch.compile`` or ``torch.export``. The runtime module does not expose a
   public C++ API — use the Python :doc:`py_api/runtime` module instead.
